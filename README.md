# Scoops2Go_SSD_2526

This repository is part of your assignment for Secure Systems Development (6G7V0030_2526_1F). It's fairly empty at the momement, but over the next 11 weeks, you'll be working as a team to build a secure system - Scoops2Go - in this very repository!

In each weekly lab session you will work within your team to make changes to this repository. You can also make changes outside of the lab sessions if you like, but be sure to do this using The GitHub Flow so that your team is kept in the loop and can review your changes (we will cover The GitHub Flow in week 2).

You will be assessed individually via a practice journal that highlights evidence of key personal attributes (such as teamworking) and specialist skills (such as knowledge of version control systems) in this very repository.

Further information can be found on the Moodle area for this module.
